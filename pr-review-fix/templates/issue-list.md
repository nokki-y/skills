# 指摘一覧テンプレート

取得したレビューコメントを以下の形式で一覧化してください。

## 一覧表形式

| # | ファイル | 行番号 | 指摘内容 | 種別 | 対応要否 |
|---|---------|--------|----------|------|----------|
| 1 | path/to/file.ts | L42 | 変数名が不適切 | 改修必須 | 要 |
| 2 | path/to/file.ts | L58 | コメント追加推奨 | 提案 | 任意 |
| 3 | path/to/api.ts | L15 | なぜこの実装にした？ | 質問 | 回答要 |
| 4 | - | - | LGTM | 承認 | - |

## 記載ルール

- **#**: 連番（対応時の参照用）
- **ファイル**: 指摘対象のファイルパス
- **行番号**: 指摘対象の行（L + 行番号）
- **指摘内容**: レビューコメントの要約（簡潔に）
- **種別**: 下記の分類に従う
- **対応要否**: 要 / 任意 / 回答要 / -
